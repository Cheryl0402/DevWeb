To Do List

Task Manage Tool

The Todo List is a customized simple tool with which I can easily manage my daily tasks. It is very simple to use. I can add new task by clicking on add button, delete task by clicking on garbage icon, and strikethrough a task when it is finished.

Here is a video demo:

Front end technologies used here are HTML, CSS, and JavaScript.

CSS Styling mainly focuses on list container and page background. It defines task input content background and text transformation as well as hover and focus movement in the container. The simple but nice-looking body background is made with linear gradient.

JavaScript is used to control task actions including adding a task, striking through a task, and deleting a task.
