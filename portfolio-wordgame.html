Guess A Word

Word Guess Game

Learning by playing.

This is a web game for learning English word while having fun. Word groups can vary depending on the vocabulary database. The hint icon gives hint according to the Word group.

The simple and responsive web game is built with HTML, CSS, and JavaScript. With JavaScript, class and function definition, flow control, and DOM manipulation are used to achieve user interaction features including starting new game, guessing and checking letters, displaying feedback messages.
